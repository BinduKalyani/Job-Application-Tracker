# Job Application Tracker

A RESTful backend application built using Java and Spring Boot for managing job applications and interview statuses.

## Features

* Retrieve all job applications
* Add new job applications
* Return structured JSON responses
* REST API architecture

## Technologies

* Java
* Spring Boot
* Maven
* REST APIs
* GitHub Codespaces

## Sample Response

```json
[
  {
    "id": 1,
    "company": "Google",
    "position": "Software Engineer",
    "status": "Applied"
  },
  {
    "id": 2,
    "company": "Amazon",
    "position": "Backend Engineer",
    "status": "Interview"
  }
]
```
